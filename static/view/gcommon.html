GCommon

GCommon is a logic manipulation environment for virtual reality simulation.

Features include primitive constructive solid geometry and navigable 3d environments.

  • Scripting engine
  • Dynamic objects
  • Serialization
  • Indexed database
  • Networked server/client authentication
  • Basic shader support

Partially supported features include:

  • Networked backed database cache
  • various physics properties (collision, gravity, friction, and momentum)
  • advanced shader properties (skin, cloth, etc)
  • modelling processes (quite advanced including predictive, chain, and on-demand calculation, as well as model scale resolution shifting which rocks socks)

Planned features include:

  • Shared client/server computation cluster
  • advanced geometries/physics (fluids, gasses, electrochemistry)
  • model/module integration (eg breeding module+life model)
  • lots of models:
    • solar model: orbitting planets, asteroids, phenomenae
    • planet model: plates, landmasses, weather
    • seed model:
    • tech model
    • civilization model
    • more...
