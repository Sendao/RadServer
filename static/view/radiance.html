Radiance

Radiance is a clientside template processor that has advanced variable manipulation and tracking capabilities which provide flexible layout and update/animation options, including intelligent and easy reshaping and sizing, multiple levels of data monitoring for automatic updates, update schedulers and dynamic variable parsing against a live global variable mesh, along with much more.

It can be found on Sourceforge or, for a more recent version check out the radserver on github, which includes the 'Reserve' webserver described below.
