Webserve

Webserve is a dynamic modular webserver written in C++. Modules are loaded as applications which then run in threads on the main process.

Features include xdb integration, dynamic datatype serialization features, static file support including binary files.

Partially developed features include SSL, intelligent caching, upload, and download retry/resume.

Considered features include ftp, scp, and ssh support behind ssl+warpkit.
