RadServer

RadServer is a nodejs webserver using journey and radjs.

Features include:

  • SSL/HTTPS
  • Websockets, including secure (ssl) sockets
    • Secure (ssl) sockets
    • Automation included (tpl/socket.html)
  • Integrated database
    • Partial database editor
    • Extensible for replication, archiving, and encryption
    • Secure (no open sockets or inter-process comms)
    • Natural (works well with node's process philosophy)
    • NoSQL principles
    • Some support for extended queries
    • Auto-loading object wrapper auto-describes and saves most objects easily
    • Very, very, very fast... I should definitely benchmark this
  • Scraping integration
    • Curl and Cheerio are included
    • Interfaces for both are available
    • There Are Plans And History
  • Dynamic Handling
    • PHP is supported, actually. No big deal!
    • Static files of all types are easily and securely handled.
    • Model-based nodejs interfaces, easily extensible and integrated.
    • Easy to configure sockets for streaming live views of database objects.

It can be found on github.
