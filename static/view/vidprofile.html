ColoGrav Java Demo

Here is an example screensaver written in Java. The core idea behind the animation is of color-mapped gravity. I wanted to examine how gravity might work if it was not unilateral. I developed a visualization process whereby objects are assigned red, green, and blue color levels, and these levels are then applied to a gravitational lensing layer. The lensing layer as well as the moving objects are strangely hypnotic in their movements.

SpiritShare Social Network

SpiritShare was a small social network I set up to pull data from Facebook and Google, searching for related items when people were posting common data. It was more of a research site than a social network, but also included the features needed to interact with facebook's content, as well as some convenient visual layouts facebook to make it an effective research tool.

SkyGrid Virtual Server Manager

SkyGrid was a virtual machine controller to manage AppLogic machines. AppLogic combines virtual machine functionality with application packaging to provide application-as-a-service (AAAs? you bet!) support for both hosting and administration purposes. Features include on-the-fly resizing, machine rebuilds, and assisted application-sensitive migrations.
